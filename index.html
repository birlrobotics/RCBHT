Probabilistic Relative Change-Based Hierarchical Taxonomy (pRCBHT)

HL State Estimation for FT Signals in Robot Contact Tasks.

View the Project on GitHub rojas70/SS_matlab_offline_coder

This code takes 6 FT signals and encodes them in an increasingly abstract manner to perform HL state estimation (behavior estimation and task output verification) for robot contact tasks. It assumes the PivotApproach Strategy (code: rojas70.github.io/PivotApproach) and 4 state FSM.

This code has used to run experiments for the following publications:

J. Rojas, WenWei Kuang, Ding Qiao, and Kensuke Harada, High Level State Estimation and Task Deliberation for Dual Armed Assembly Tasks, IEEE International Conference on Intelligent Robots and Systems (IROS), (under-review), Daejon, Korea, Oct 9-14, 2016 (SCI/EI).
